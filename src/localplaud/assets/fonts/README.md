# Bundled export font

`NotoSansTC.ttf` is the variable Noto Sans Traditional Chinese font from the
[Google Fonts repository](https://github.com/google/fonts/tree/main/ofl/notosanstc).
It is bundled so exported PDFs embed the glyphs required for portable Taiwan
Mandarin and mixed Mandarin/English transcripts.
